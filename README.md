# 💰 Expense Tracker (TypeScript)

A modern and responsive **Expense Tracker Web App** built using **TypeScript, HTML, and CSS** to manage daily income and expenses بسهولة واحترافية.

---

## 🚀 Features

* ✅ Add transactions (Income / Expense)
* ✅ Real-time balance calculation
* ✅ Dynamic UI updates
* ✅ Delete transactions
* ✅ Clean and responsive design
* ✅ TypeScript strict typing
* ✅ DOM manipulation with type safety

---

## 🛠️ Built With

* TypeScript
* HTML5
* CSS3
* DOM API
